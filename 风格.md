# 网站视觉与交互风格约束

本文档是本站 UI 的**单一事实来源**：新页面与改版以本文为准，并与 `src/index.css` 中的设计令牌保持一致。

---

## 1. 设计原则

- **令牌优先**：颜色、圆角、间距优先使用 `@theme` 与既有 `mizuno-*` 色阶，避免页面内随意新增十六进制色值。
- **层次清晰**：页面背景、内容区、卡片/浮层分为不同表面亮度，用边框与圆角区分，少用重阴影。
- **可读性**：正文以衬线体为主（`font-serif`），导航、按钮、标签等界面文案用无衬线（`font-sans`）。
- **克制动效**：过渡时长约 150–250ms；尊重 `prefers-reduced-motion`（已在全局 CSS 处理）。
- **可访问性**：交互元素具备 `:focus-visible` 轮廓；图标装饰需配合可见文字或 `aria-label`。

---

## 2. 色彩与表面（Tokens）

| 用途 | 约定 |
|------|------|
| 品牌主色 | `mizuno-600`（浅色模式强调、链接）；深色模式头部等可用 `mizuno-800` |
| 页面背景 | `bg-mizuno-50` / `dark:bg-mizuno-900` |
| 主文字 | `text-mizuno-900` / `dark:text-mizuno-50` |
| 次要文字 | `text-mizuno-800` / `dark:text-mizuno-200` |
| 辅助说明 | `text-mizuno-600` / `dark:text-mizuno-400` |
| 卡片/分区表面 | `bg-white/80` 或 `bg-mizuno-100/60`，深色 `dark:bg-mizuno-800/50` |
| 卡片边框 | `border-mizuno-200` / `dark:border-mizuno-700` |
| 焦点环 | `outline-mizuno-600`，与品牌一致 |

---

## 3. 字体

- **正文**：`font-serif`（Georgia 系栈，见 `index.css` `--font-serif`）。
- **UI**（导航、面包屑、按钮、卡片标题）：`font-sans`（系统无衬线栈）。
- **标题层级**：页面主标题 `text-3xl font-semibold tracking-tight`；区块标题 `text-xl font-semibold`。

---

## 4. 布局与间距

- 主内容宽度与站点一致：`max-w-3xl`，水平 `px-4`。
- 区块纵向节奏：标题下 `mb-3`～`mb-4`，段落 `mb-6`～`mb-8`。
- **Fun 入口**：分类使用网格卡片；窄屏单列，宽屏可 `sm:grid-cols-3`（三列等宽）。

---

## 5. 组件与交互模式

- **链接**：`underline underline-offset-2`；hover 略加深颜色；`focus-visible:outline` 2px。
- **卡片式导航（Fun 分类）**：
  - 整卡可点击（`Link` 包裹或 `relative` + 伸长的伪链接），hover 时边框略亮、可选轻微 `translate-y` 或背景变化。
  - 圆角：`rounded-xl`；内边距：`p-5`；图标区与文案区横向排列（`flex gap-4`）。
- **面包屑**：`font-sans text-sm`，颜色用辅助色阶，末级不加下划线。

---

## 6. Fun 专区：分类与图标令牌

Fun 下分三类；**每一类有固定图标令牌**（实现上统一尺寸 `size={28}`、`strokeWidth={1.75}`，保持线宽一致）。

| 分类 | 路由 | 图标令牌 | 说明 |
|------|------|----------|------|
| **Games** | `/fun/games` | `Gamepad2`（Lucide） | 手柄/控制器意象 |
| **Tools** | `/fun/tools` | `Hammer`（Lucide） | 工具、实用程序 |
| **Sports** | `/fun/sports` | `BadmintonIcon`（站内 SVG 组件） | 羽毛球 + 球拍意象 |

分类页（Games / Tools / Sports）的页面标题行应在标题旁或上方一致展示对应图标，强化识别。

Sports 页当前可为占位内容；新增具体运动子页时，在本表追加一行子路由与图标策略。

---

## 7. 图标通用规则

- **线框风格**：与 Lucide 一致，避免混用填充风图标。
- **尺寸**：卡片内分类图标约 28px；内联小图标 18–20px。
- **颜色**：默认 `text-mizuno-600 dark:text-mizuno-400`，置于 `bg-mizuno-100 dark:bg-mizuno-800/80` 的圆角容器内（如 `rounded-lg p-2`）。

---

## 8. Markdown 内容区

博客与文章使用 `prose-mizuno` 类族（见 `index.css`），不在此重复；新元素若需加入 prose，应同步更新 `index.css` 中的 `.prose-mizuno` 规则以保持对比度。

---

## 9. 修订

变更令牌或 Fun 分类图标时，请同时更新：`src/index.css`、相关组件与本文档。
